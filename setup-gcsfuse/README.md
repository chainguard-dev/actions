# Setup `gcsfuse`

This action installs the latest `gcsfuse` binary for a particular environment.

```yaml
- uses: chainguard-dev/actions/setup-gcsfuse@0cda751b114eb55c388e88f7479292668165602a # v1.0.2
```

## Scenarios

```yaml
steps:

```
